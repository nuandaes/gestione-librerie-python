# Gestione Librerie Python

Un programma Python con interfaccia grafica per:

- Installare librerie.
- Aggiornare librerie.
- Disinstallare librerie.
- Verificare aggiornamenti di Python e pip.

## Requisiti

- Python 3.8+
- Moduli richiesti:
  - `tkinter`
  - `requests`
  - `urllib`
  - `shutil`

## Esecuzione

Avvia il programma eseguendo:

```bash
python "PythonLibs v3.py"
